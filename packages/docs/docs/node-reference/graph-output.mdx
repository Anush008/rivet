## Overview

The Graph Output Node defines an output for a graph. The outputs are returned after processing a graph via the SDK, or defines one of the output ports when the graph is used as a subgraph.

## Editor Settings

| Setting                 | Description                                                                                                                         | Default Value | Notes |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------- | ------------- | ----- |
| ID                      | The ID of the output. This will be used to identify the output when the graph is called, and defines the name of the output in the UI. | (required)    | N/A   |
| Data Type               | The data type of the output.                                                                                                        | `string`      | N/A   |

</TabItem>

## Example 1: Define outputs for a graph

1. Create a new graph and add two Graph Output Nodes to it.
2. In the Editor Settings, set the first Graph Output Node's ID to `firstOutput` and the Data Type to `string`.
3. In the Editor Settings, set the second's ID to `secondOutput` and the Data Type to `string`.
4. Create a new graph, and add a Subgraph Node. Note the `firstOutput` and `secondOutput` nodes.
